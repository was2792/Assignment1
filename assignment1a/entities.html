This is a simple file that will show some entitys.


Because you use certain symbols such as < & > to write in HTML
They are given special entity names so that they still may be used!
You can also use entity numbers to write certain symbols such as:


¢ - Which is the symbol for cents
£ - Which is a pound symbol
¥ - Which is the symbol for Yen
© - Which is the symbol for a copyright
